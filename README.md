## 📌 Project Overview

This project is a simple Python-based tool that helps identify whether a website URL is safe or potentially a phishing attempt. It is designed as part of a BYOP (Bring Your Own Project) to solve a real-life cybersecurity problem.

## ❗ Problem

Many users unknowingly enter sensitive information on fake websites that look similar to real ones. These phishing sites can steal passwords, banking details, and personal data.

## 💡 Solution

This program checks a given URL and analyzes it based on certain patterns such as:

* Use of HTTPS
* Length of the URL
* Presence of special characters like '@' and '-'
* Suspicious words like "login", "verify", "bank"

Based on these checks, the system decides whether the website is safe or suspicious.

## ⚙️ Technologies Used

* Python (basic programming)
* Simple AI logic (pattern-based detection)

## ▶️ How to Run

1. Make sure Python is installed
2. Download or clone this repository
3. Open terminal in the project folder
4. Run the file:

python phishing_detector.py

## 🧪 Example

Input:
http://login-bank-secure.com

Output:
This site looks unsafe
